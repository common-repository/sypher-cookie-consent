Sypher cookie consent banner integration for Wordpress

== Description ==

Choose the Sypher Cookie Consent Manager to improve your data privacy practices and help your organization with the
ePrivacy Directive and GDPR. The EU legislation requires consent for the placement of cookies and a lawful basis for
certain processing of the data from cookies stored on users devices (computers, laptops, mobile phones or tablets).

This module offers a number of features designed to make the compliance process easier for the display of cookie
banners, appropriate placement of cookies and capturing consent.

The Sypher Cookie Consent for Wordpress makes integration with the Sypher Cookie Consent Manager as simple as
activating the plugin and providing your website UUID key.

== Installation ==

1. Upload the `sypher-cookie-consent` directory to the `wp-content/plugins` directory
2. Activate the plugin through the `Plugins` menu in WordPress
3. Adjust the settings on the `Sypher Cookie Consent` Settings page
